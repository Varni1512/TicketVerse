# Security Policy

## Supported Versions

Only the latest major version (currently v1.0.x) receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within TicketVerse, please send an e-mail directly to the project maintainers rather than creating a public issue.

We will review all vulnerability reports and work with you to resolve them in a timely manner. We ask that you do not share or publish the vulnerability until we have had a chance to fix it.
